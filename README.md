# PostgreSQL-Queries
This repository contains the English version of the SQL assignment provided by Patika.dev. 
The assignment focuses on querying the DVD Rental database using SELECT statements, filtering conditions, and logical operators.

## Files
- dvd_rental_queries.sql : Contains all SQL queries for Assignment 1
- sql_assignment2.sql : Contains all SQL queries for Assignment 2

## Notes
- All queries are written in standard SQL and tested on the PostgreSQL DVD Rental sample database.
- Logical conditions and filtering are included using AND, OR, NOT, and IN operators.
